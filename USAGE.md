# 使用指南

## 访问应用

开发服务器已启动，请在浏览器中访问：**http://localhost:5173/**

## 功能导航

### 1. 首页 (/)
- 查看平台简介和核心功能
- 使用快捷入口快速访问常用功能

### 2. AIGC 内容生成

#### Deepfake 人脸生成 (/generate/deepfake)
1. 上传目标人脸图片
2. 上传驱动人脸/视频
3. 选择功能类型（人脸替换/动画/属性编辑）
4. 选择对应的模型
5. 点击"开始生成"按钮
6. 等待生成完成后可预览和下载结果

#### 多模态内容生成 (/generate/general)
**图像生成标签页：**
1. 输入描述性提示词
2. 选择图像尺寸
3. 点击"生成图片"
4. 预览和下载生成的图像

**视频生成标签页：**
1. 输入描述性提示词
2. 可选上传参考图片
3. 选择生成时长
4. 点击"生成视频"
5. 预览和下载生成的视频

### 3. AIGC 内容检测

#### 虚假内容检测 (/detect/fake)
1. 点击或拖拽上传图片/视频
2. 系统自动进行检测
3. 查看检测结果：
   - 真实/虚假判断
   - 伪造概率
   - 可疑区域标注
   - 检测模型信息
4. 可导出检测报告

#### 不安全内容检测 (/detect/unsafe)
1. 上传待检测的图片/视频
2. 查看检测结果：
   - 风险等级（低/中/高）
   - 违规类型列表
   - 详细分析
   - 整改建议
3. 可导出检测报告

### 4. 数据流转

#### 数据接入 (/data/input)
**本地上传标签页：**
1. 拖拽或选择文件上传
2. 查看上传进度
3. 支持暂停/继续上传
4. 支持批量上传

**网络爬取标签页：**
1. 配置平台授权（YouTube/Bilibili 等）
2. 输入目标 URL
3. 点击"开始爬取"
4. 查看爬取进度（连接→下载→预处理）

#### 数据输出 (/data/output)
**生成内容标签页：**
1. 浏览所有生成的图像和视频
2. 点击"预览"查看完整内容
3. 点击"下载"配置下载选项
4. 支持批量导出

**检测记录标签页：**
1. 查看所有检测历史
2. 包含虚假检测和安全检测记录
3. 显示检测结果和置信度

## 技术说明

### Mock 数据
- 所有 API 请求都被 MSW 拦截
- 返回预设的模拟数据
- 不会发送真实网络请求
- 模拟了网络延迟（500ms-2s）

### 占位资源
- 图像使用 picsum.photos 占位服务
- 生成结果使用静态 mock 图片
- 热力图使用 CSS 渐变模拟

### 浏览器兼容性
- 推荐使用 Chrome/Edge/Firefox 最新版本
- 需要支持 ES2020+ 特性
- 需要支持 Service Worker

## 常见问题

**Q: 为什么上传后没有真正的检测结果？**
A: 本项目使用 Mock 数据，所有结果都是预设的模拟数据，用于演示功能和界面。

**Q: 如何停止开发服务器？**
A: 在终端按 Ctrl+C 停止服务器。

**Q: 如何连接真实后端？**
A: 需要修改 `src/main.tsx` 移除 MSW 配置，并在 `src/utils/request.ts` 中配置真实的 API 地址。

**Q: 页面加载很慢？**
A: 首次加载需要初始化 Service Worker，刷新页面即可恢复正常速度。

## 下一步开发

如需对接真实后端：
1. 修改 `src/main.tsx`，移除或条件化 MSW 启动代码
2. 在 `src/utils/request.ts` 中配置真实 API baseURL
3. 根据后端接口规范调整请求和响应的数据结构
4. 添加真实的身份认证和授权逻辑
